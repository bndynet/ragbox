# Widget Theming

The widget supports a compact set of presentation tokens so docs sites can match their existing design system.

```mdx
<RagboxWidget source="docs" theme="light" />
```

## Tokens

- `--ragbox-accent`: primary action color.
- `--ragbox-surface`: panel background.
- `--ragbox-text`: primary text color.
- `--ragbox-border`: border color.

## Accessibility

Use color pairs with enough contrast for body text and interactive controls. Preserve visible focus styles for the input field and submit button.
